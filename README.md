# \<tpx-modal>

This webcomponent follows the [open-wc](https://github.com/open-wc/open-wc) recommendation.

## Installation
```bash
npm i tpx-modal
```

## Usage
```html
<script type="module">
  import 'tpx-modal/tpx-modal.js';
</script>

<tpx-modal></tpx-modal>
```
